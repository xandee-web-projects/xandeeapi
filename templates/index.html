What is a ReST API?

REST, short for REpresentational State Transfer is a way to dynamically create and update data In your apps without having to make updates

How it works

In a ReST API, There are basically 4 operations that can be carried out which are know as CRUD operations which stand for CREATE, READ, UPDATE, DELETE which are already self explanatory. This allows for remote creating, reading, updating and deleting data in a database

What we offer

Xandee ReST API offers a way to host files and call and update them over the api. Which means you can create or change variables, make updates, allow clients to make updates (Eg. In a leaderboard system) for your websites or apps without having to rebuild them or edit them directly. The best way to look at it is like environment variables but can be edited easily.
